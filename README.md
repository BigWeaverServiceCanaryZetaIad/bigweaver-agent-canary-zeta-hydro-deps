# bigweaver-agent-canary-zeta-hydro-deps

This repository contains heavyweight dependencies and benchmarks for the bigweaver-agent-canary-hydro-zeta project.

## Contents

### Benchmarks (`benches/`)

Performance benchmarks for Hydro and related crates, including:
- Timely and differential-dataflow benchmarks
- Various dataflow operation benchmarks (reachability, join, fan_in, fan_out, etc.)

See [benches/README.md](benches/README.md) for more details on running benchmarks.

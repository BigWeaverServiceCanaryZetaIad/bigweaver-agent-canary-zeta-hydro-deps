# bigweaver-agent-canary-zeta-hydro-deps

This repository contains benchmark dependencies for the bigweaver-agent-canary-hydro-zeta project.

## Benches

The `benches` directory contains performance benchmarks for comparing DFIR (Hydroflow) implementations against timely-master and differential-dataflow-master frameworks.

See [benches/README.md](benches/README.md) for more information about running the benchmarks.

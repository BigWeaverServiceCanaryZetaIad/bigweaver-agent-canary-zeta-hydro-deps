# Microbenchmarks

Of Hydro and other crates (timely-dataflow and differential-dataflow).

## Quick Reference

Run all benchmarks:
```bash
cargo bench
```

Run specific benchmarks:
```bash
cargo bench --bench reachability
cargo bench --bench arithmetic
```
